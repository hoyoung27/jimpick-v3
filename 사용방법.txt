짐픽 V3.5 자동 차량·인원 추천

추가 기능
- 사진과 동영상 최대 50개 업로드
- 업로드 개수 표시
- 사진 분석 도우미 품목 수량 ＋ / − 조절
- 선택 품목 총 작업량 표시
- 예상 짐의 양 표시
- 1톤·5톤 차량 자동 추천
- 작업 인원 자동 추천
- 품목 반영 전 결과 미리보기

사용 방법
1. STEP 3에서 사진 또는 동영상을 추가합니다.
2. 사진 분석 도우미 열기를 누릅니다.
3. 품목별 ＋ / − 버튼으로 수량을 입력합니다.
4. 차량·인원 추천을 확인합니다.
5. 품목에 반영을 누릅니다.
