짐픽 PRO 6.3.2 DEMO - 화면 클릭 긴급수정

수정 내용
- 체험 화면 위에 남아 있던 투명 클릭 차단막 제거
- 로그인창·구독창·로딩창 강제 닫기
- 견적 화면의 버튼·입력칸·상단 단계 버튼 클릭 복구
- 체험 화면 진입 후 클릭 상태를 한 번 더 자동 점검

적용 순서
1. 압축을 풉니다.
2. GitHub에 전체 파일을 덮어씁니다.
3. Netlify에서 Deploy project without cache를 실행합니다.
4. 사이트에서 Ctrl + Shift + R을 누릅니다.
5. 왼쪽 위 PRO 6.3.2 DEMO 확인
6. 로그인 없이 견적 화면 보기 클릭
7. 상단 단계 버튼과 화면 안 버튼이 눌리는지 확인
