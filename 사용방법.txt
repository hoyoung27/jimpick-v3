짐픽 V4.8 매출·순이익 통계

추가 기능
- 사장님 대시보드에 매출통계 탭
- 월별 총 견적금액
- 월별 총 입금액
- 월별 미수금
- 월별 예상 순이익
- 문의/견적중/계약완료/작업완료/취소 건수
- 최근 6개월 실적표
- 선택 월 상세 고객내역
- CSV 매출내역 다운로드

사용 위치
고객정보 → 사장님 대시보드 → 매출통계

업로드 방법
1. ZIP 다운로드
2. 압축 풀기
3. GitHub jimpick-v3 → Add file → Upload files
4. 폴더 안 파일 6개 모두 업로드
5. Commit changes
6. Netlify Published 확인
7. Ctrl + Shift + R 또는 Ctrl + F5
