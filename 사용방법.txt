짐픽 PRO 4.1 - 현장 안정화판

이번 핵심 작업
- 고객정보와 견적 자동저장
- 화면을 닫아도 최근 견적 복원
- 인터넷 연결상태 표시
- 서버 저장 실패 시 기기에 임시보관
- 인터넷 복구 후 대기자료 다시 저장
- 저장 대기 건수 표시
- AI 분석 실패 시 재시도 안내
- 기존 AI 완전자동견적, 구독, 결제, 공유 기능 유지

시험 방법
1. 고객정보와 품목을 입력합니다.
2. 브라우저를 닫았다 다시 엽니다.
3. 상단 복원 안내에서 '복원'을 누릅니다.
4. 인터넷을 끄고 서버 저장을 누릅니다.
5. 다시 인터넷을 연결한 뒤 '저장 대기자료 다시 보내기'를 누릅니다.

업로드
1. 압축 풀기
2. GitHub에 전체 파일과 netlify, supabase 폴더 업로드
3. Commit changes
4. Netlify에서 Deploy project without cache
5. Ctrl + Shift + R
