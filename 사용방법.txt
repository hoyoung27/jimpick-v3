짐픽 PRO 6.1 BETA - 현장테스트·오류수집

이번 핵심 작업
- 현장 베타테스트 시작·종료
- 로그인·사진·AI·견적·가격·문자·서명·내역 단계별 체크
- 문제와 불편사항 기록
- 기기·브라우저·인터넷 상태 자동 기록
- 오류정보 한 번에 복사
- 베타테스트 결과 Supabase 저장
- 기존 출시 자동점검과 모든 견적 기능 유지

사용 순서
1. 로그인 후 상단 '현장테스트' 클릭
2. 테스트 시작
3. 실제 견적 흐름을 끝까지 사용
4. 정상 작동한 항목 체크
5. 불편하거나 틀린 점 입력
6. 테스트 결과 저장
7. 오류가 있으면 오류정보 복사 후 전달

반드시 할 일
Supabase SQL Editor에서 supabase/setup.sql을 다시 실행하세요.

업로드
1. 압축 풀기
2. GitHub에 전체 파일과 netlify, supabase 폴더 업로드
3. Commit changes
4. Netlify에서 Deploy project without cache
5. Ctrl + Shift + R
