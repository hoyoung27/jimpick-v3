짐픽 PRO 6.3.3 DEMO - 단계 버튼 긴급수정

수정 내용
- 고객정보 버튼 직접 연결
- 사진촬영 버튼 직접 연결
- 품목확인 버튼 직접 연결
- 자동견적 버튼 직접 연결
- 문자발송 버튼 직접 연결
- 이전·다음 버튼 체험모드 이동 복구
- 기존 프로그램 초기화가 실패해도 단계 이동 가능

적용 순서
1. 압축을 풉니다.
2. GitHub에 전체 파일을 덮어씁니다.
3. Netlify에서 Deploy project without cache 실행
4. Ctrl + Shift + R
5. 왼쪽 위 PRO 6.3.3 DEMO 확인
6. 로그인 없이 견적 화면 보기 클릭
7. 상단 5개 단계 버튼을 각각 클릭
