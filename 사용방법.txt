짐픽 PRO 5.0 - 업체별 요금·대표 최종결정

핵심 원칙
- AI는 가격을 강제로 정하지 않습니다.
- AI는 짐량, 차량, 작업인원, 작업시간만 계산합니다.
- 업체별 요금표를 이용해 참고금액을 계산합니다.
- 최종 견적금액은 각 업체 대표가 직접 결정합니다.

사용 순서
1. 업체설정에서 자기 업체 요금표 입력
2. 사진 AI 자동분석
3. 차량·인원·시간 확인
4. 업체 요금표 계산금액 확인
5. 할인금액 또는 추가 조정금액 입력
6. 대표님 최종금액 적용
7. 견적서 공유 또는 문자 발송

예시
업체 계산금액 1,000,000원
할인금액 100,000원
추가 조정금액 0원
최종 견적 900,000원

업로드
1. 압축 풀기
2. GitHub에 전체 파일과 netlify, supabase 폴더 업로드
3. Commit changes
4. Netlify에서 Deploy project without cache
5. Ctrl + Shift + R
