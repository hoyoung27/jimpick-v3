짐픽 PRO 6.0 RC - 출시 후보 자동점검판

이번 핵심 작업
- 로그인 서버 자동점검
- 사진 AI 서버 자동점검
- 업체명과 차량요금 입력 여부 확인
- 토스 결제 설정 확인
- 기기 자동저장 확인
- 앱 설치·자동업데이트 환경 확인
- 정상 항목과 확인 필요 항목 구분
- 문제가 있는 항목만 빨간색으로 안내
- 기존 AI 견적·요금표·서명·계약·문자 기능 유지

사용 순서
1. 로그인
2. 고객정보 화면의 '출시 준비 자동 점검' 클릭
3. 빨간색 확인 필요 항목 확인
4. 해당 설정만 수정
5. 모두 정상으로 표시되면 실제 휴대전화와 태블릿에서 마지막 시험

업로드
1. 압축 풀기
2. GitHub에 전체 파일과 netlify, supabase 폴더 업로드
3. Commit changes
4. Netlify에서 Deploy project without cache
5. Ctrl + Shift + R
