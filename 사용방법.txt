짐픽 PRO 2.1 - 현장시험과 AI 연결점검

이번 핵심 작업
- 로그인 화면에 '로그인 없이 기능 시험하기' 추가
- 이메일 입력 예시를 실제 이메일 형식으로 수정
- 로그인 서버 연결상태 점검
- Gemini 사진 AI 연결상태 점검
- 빠진 환경변수를 한글로 안내
- 시험 모드에서 고객정보를 자동 입력
- 사진 촬영 → AI 분석 → 자동견적 → 문자 흐름 직접 시험

시험 방법
1. 로그인 화면에서 '로그인 없이 기능 시험하기' 클릭
2. 고객정보 화면에서 다음 클릭
3. 사진 AI 화면에서 '지금 점검' 클릭
4. 두 항목이 모두 '정상'인지 확인
5. 사진을 찍고 AI 자동분석 시작

주의
시험 모드에서는 서버 견적 저장과 실제 회원가입이 되지 않습니다.

업로드
1. 압축 풀기
2. GitHub에 전체 파일과 netlify, supabase 폴더 업로드
3. Commit changes
4. Netlify → Deploy project without cache
5. Ctrl + Shift + R

필수 환경변수
- SUPABASE_URL
- SUPABASE_ANON_KEY
- GEMINI_API_KEY

정상 표시
build JIMPICK-PRO-2.1
