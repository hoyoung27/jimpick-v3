짐픽 PRO 2.0 - AI 자동견적 완성 흐름

핵심 사용 순서
1. 로그인
2. 고객정보 입력
3. 사진 최대 24장 촬영/선택
4. AI 자동 품목 인식
5. 품목 확인
6. 1톤·5톤 차량, 인원, 시간, 금액 자동계산
7. 견적 확정·서버 저장
8. 고객 문자 보내기

문자 보내기
- Android/아이폰/태블릿: 기본 문자 앱 자동 실행
- PC: 문자 내용 자동 복사

업로드
1. 압축 풀기
2. 전체 파일과 netlify, supabase 폴더를 GitHub에 업로드
3. Commit changes
4. Netlify → Deploy project without cache
5. Ctrl + Shift + R

필수 환경변수
- SUPABASE_URL
- SUPABASE_ANON_KEY
- GEMINI_API_KEY

정상 표시
build JIMPICK-PRO-2.0
