짐픽 PRO 5.7 - 고객안내문자·잔금요청

이번 핵심 작업
- 견적 확정 문자 자동작성
- 계약금 입금 안내 자동작성
- 이사 전날 안내 자동작성
- 잔금 요청 문자 자동작성
- 완납 확인 문자 자동작성
- 업체명·고객명·이사일·금액 자동입력
- 휴대전화 문자 앱 바로 열기
- 문자 내용 복사
- 계약관리 화면에서 고객 문자 바로 보내기

사용 순서
1. 최종 견적과 계약금 입력
2. 원하는 안내 종류 선택
3. 자동 작성된 문자 확인
4. '문자 앱 열기' 클릭
5. 휴대전화 문자 앱에서 전송

업로드
1. 압축 풀기
2. GitHub에 전체 파일과 netlify, supabase 폴더 업로드
3. Commit changes
4. Netlify에서 Deploy project without cache
5. Ctrl + Shift + R

정상 표시
