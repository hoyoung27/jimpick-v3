짐픽 PRO 5.1 - 선택형 요금표

이번 핵심 작업
- 차량 요금은 기본 사용
- 작업자 요금 사용 여부 선택
- 추가 작업시간 요금 사용 여부 선택
- 사다리차 요금 사용 여부 선택
- 특수품목 요금 사용 여부 선택
- 분해작업 요금 사용 여부 선택
- 중량품목 요금 사용 여부 선택
- 사용하지 않는 요금은 계산에서 제외
- 최종 견적금액은 대표가 직접 결정

사용 순서
1. 로그인 후 업체설정 클릭
2. '견적에 포함할 요금 선택'에서 필요한 항목만 켜기
3. 켠 항목의 금액 입력
4. 저장
5. 다음 견적부터 선택한 항목만 자동 계산
6. 마지막에 대표가 할인·추가금액을 조정해 최종 가격 확정

중요
작업자 요금과 추가 작업시간 요금은 기본적으로 꺼져 있습니다.
필요한 업체만 직접 켜서 사용할 수 있습니다.

반드시 할 일
Supabase SQL Editor에서 supabase/setup.sql을 다시 실행하세요.

3. Commit changes
