짐픽 PRO 6.3.4 DEMO - 로그인 없는 완전 체험판

체험 가능한 기능
- 고객정보 화면 이동과 입력
- 사진촬영·갤러리 버튼으로 예시 사진 등록
- 사진 AI 분석으로 예시 품목 표시
- 품목 수량 수정과 삭제
- 차량·인원·시간 자동견적
- 업체 계산금액·할인·최종금액 수정
- 견적확정·계약금·잔금 계산
- 안내문자 종류 선택
- 문자내용 복사
- 고객 서명 연습

체험모드에서 제한되는 기능
- 실제 서버 저장
- 실제 결제
- 실제 문자전송
- 실제 AI API 사용

적용 순서
1. 압축을 풉니다.
2. GitHub에 전체 파일을 덮어씁니다.
3. Netlify에서 Deploy project without cache 실행
4. Ctrl + Shift + R
5. 왼쪽 위 PRO 6.3.4 DEMO 확인
6. 로그인 없이 견적 화면 보기 클릭
