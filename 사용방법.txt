짐픽 PRO 7.6 RC - 방·품목 드래그 관리

추가 기능
- 방 추가
- 방 삭제
- 방 이름 변경
- 방 순서 드래그 변경
- 품목 순서 드래그 변경
- 품목을 다른 방으로 드래그 이동
- 품목 수량 조절
- 품목 삭제
- 즐겨찾기 품목
- 즐겨찾기 품목을 선택 방에 빠르게 추가
- 변경내용 자동저장
- 기존 STEP 3 데이터와 자동 동기화
- 기존 7.5 AI 사진분석 기능 유지

사용
1. 방 카드 빈 공간을 잡고 드래그하면 방 순서가 바뀝니다.
2. 품목 카드를 다른 방으로 드래그하면 이동합니다.
3. 방 카드의 연필 버튼으로 이름을 바꿉니다.
4. 방 카드의 × 버튼으로 방을 삭제합니다.
5. 품목의 ☆ 버튼으로 즐겨찾기에 등록합니다.

적용
1. 압축 해제
2. GitHub 전체 파일 덮어쓰기
3. Netlify Deploy project without cache
4. Ctrl + Shift + R
5. PRO 7.6 RC 확인
