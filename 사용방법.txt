짐픽 PRO 6.4.1 DEMO - 화면폭·금액·다음버튼 수정

수정한 문제
- 자동견적 화면 오른쪽 빈 공간 제거
- 자동견적·문자 화면 전체 폭 사용
- 최종 견적금액 0원 문제 수정
- 예시 최종금액 980,000원 유지
- 계약금 200,000원 / 잔금 780,000원 자동 계산
- 대표님 최종금액 적용 버튼 복구
- 아래 다음 버튼을 문자발송 화면에 직접 연결
- 실시간 견적 요약 카드 추가

적용 순서
1. 압축을 풉니다.
2. GitHub에 전체 파일을 덮어씁니다.
3. Netlify에서 Deploy project without cache 실행
4. Ctrl + Shift + R
5. 왼쪽 위 PRO 6.4.1 DEMO 확인
6. 로그인 없이 견적 화면 보기 클릭
