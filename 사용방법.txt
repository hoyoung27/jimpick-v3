짐픽 PRO 2.4 - 사진 AI 완전자동견적

사용 순서
1. 로그인 없이 기능 시험하기
2. 고객정보 확인
3. 사진 촬영 또는 여러 장 선택
4. '사진 AI 완전자동견적 시작' 버튼 클릭
5. AI 품목 인식
6. 1톤·5톤 차량 자동계산
7. 작업 인원·시간·추천 금액 자동계산
8. 결과 화면 자동 이동
9. 견적 저장 또는 문자 발송

이번 핵심 작업
- 사진 분석과 자동견적을 버튼 한 번으로 실행
- 4단계 진행 흐름 표시
- 자동견적 완료 후 결과 화면으로 자동 이동
- 최종 결과를 큰 글씨로 표시
- 서버 저장과 문자 발송 버튼 연결

업로드
1. 압축 풀기
2. GitHub에 전체 파일과 netlify, supabase 폴더 업로드
3. Commit changes
4. Netlify → Deploy project without cache
5. Ctrl + Shift + R

필수 환경변수
- SUPABASE_URL
- SUPABASE_ANON_KEY
- GEMINI_API_KEY

정상 표시
build JIMPICK-PRO-2.4
