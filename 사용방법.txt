짐픽 PRO 5.8 - 사진 AI 품목 인식 강화

이번 핵심 작업
- 거실·주방·안방·작은방·베란다 촬영 안내
- 사진 여러 장 자동 분석
- 같은 물건 중복 정리 강화
- 품목별 수량과 AI 신뢰도 표시
- 신뢰도 낮은 품목 별도 표시
- 틀린 품목 삭제
- 수량 직접 수정
- 빠진 품목 직접 추가
- 수정 후 차량·인원·시간 다시 계산
- 업체별 요금표와 대표 최종가격 방식 유지

사용 순서
1. 방별 사진을 여러 장 촬영
2. '사진 AI 품목 자동인식 시작' 클릭
3. AI가 품목과 수량 분석
4. 확인 필요한 품목만 수정
5. 빠진 품목이 있으면 직접 추가
6. 차량·인원 다시 계산
7. 대표가 최종 견적금액 결정

필수 환경변수
- GEMINI_API_KEY
- SUPABASE_URL
- SUPABASE_ANON_KEY

업로드
1. 압축 풀기
2. GitHub에 전체 파일과 netlify, supabase 폴더 업로드
3. Commit changes
4. Netlify에서 Deploy project without cache
5. Ctrl + Shift + R

정상 표시
