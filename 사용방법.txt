짐픽 PRO 4.2 - 자료보호·회원탈퇴·백업

이번 핵심 작업
- 업체 프로필과 견적자료 내려받기
- 학습자료와 베타평가 자료 내려받기
- 촬영사진 기기에서 정리
- 로그인 서버와 사진 AI 서버 점검
- 비밀번호 재확인 후 계정 탈퇴
- 탈퇴 시 프로필·견적·학습·평가·결제기록 삭제
- 기존 AI 자동견적과 결제 기능 유지

사용법
1. 로그인 후 상단 '자료보호' 클릭
2. 서버 연결상태 확인
3. '내 자료 내려받기'로 JSON 백업
4. 필요하면 '촬영사진 정리'
5. 계정 탈퇴는 비밀번호를 다시 입력한 뒤 진행

필수 환경변수
- SUPABASE_URL
- SUPABASE_ANON_KEY
- SUPABASE_SERVICE_ROLE_KEY
- GEMINI_API_KEY

업로드
1. 압축 풀기
2. GitHub에 전체 파일과 netlify, supabase 폴더 업로드
3. Commit changes
4. Netlify에서 Deploy project without cache
5. Ctrl + Shift + R

정상 표시
build JIMPICK-PRO-4.2
