짐픽 PRO 5.6 - 계약내역·잔금관리

이번 핵심 작업
- 저장한 계약 목록 보기
- 고객명·전화번호·주소 검색
- 이사 날짜 검색
- 상담중·계약확정·취소·잔금완납 상태 검색
- 계약금과 잔금 확인
- 받을 잔금 합계 표시
- 잔금 완납 처리
- 계약서 다시 열기
- 계약상태 변경
- 계약자료 삭제
- 업체별 자기 계약만 표시

사용 순서
1. 로그인 후 상단 '계약관리' 클릭
2. 고객명·전화번호·날짜 또는 상태로 검색
3. 계약서를 다시 열어 확인
4. 잔금을 받았으면 '잔금 완납' 클릭
5. 필요하면 계약확정 상태 변경 또는 삭제

반드시 할 일
Supabase SQL Editor에서 supabase/setup.sql을 다시 실행하세요.

업로드

3. Commit changes
