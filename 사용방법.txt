짐픽 PRO 3.0 - 출시 준비판

이번 핵심 작업
- 앱 설치 버튼
- 새 버전 자동 감지
- 사용자 업데이트 안내
- 개인정보처리방침 초안
- 이용약관 초안
- 출시 전 점검 목록
- 오류 진단정보 복사
- 운영자·사업자 정보 입력 자리

사용법
1. 로그인 후 상단 '출시점검' 클릭
2. 앱 설치 시험
3. 점검 목록 확인
4. 개인정보처리방침과 이용약관 확인
5. 오류가 나면 '오류정보 복사' 후 전달

중요
개인정보처리방침과 이용약관은 현재 초안입니다.
실제 출시 전 운영자 정보와 환불규정을 넣고 법률 검토가 필요합니다.

업로드
1. 압축 풀기
2. GitHub에 전체 파일과 netlify, supabase 폴더 업로드
3. Commit changes
4. Netlify에서 Deploy project without cache
5. Ctrl + Shift + R

정상 표시
build JIMPICK-PRO-3.0
