짐픽 PRO 7.7 SIMPLE - 50대 쉬운 버전

개발 기준
- 설명 없이 바로 사용할 수 있는 화면
- 고객정보 → 사진 → 품목 → 금액 → 문자발송 5단계
- 같은 기능과 중복 화면 제거
- 드래그 기능 제거
- 복잡한 AI 사진분석 화면 제거
- 관리자 점수·수익분석·현장분석 기본 화면에서 숨김
- 큰 글씨, 큰 입력칸, 큰 버튼
- 컬러풀하지만 고급스러운 디자인

품목 선택
- 거실
- 주방
- 안방
- 작은방
- 기타/베란다
- 품목 수량은 − / ＋ 버튼만 사용
- 자동저장
- 1톤·5톤 차량과 작업자 추천은 자동으로 계산

업로드
1. 압축을 풉니다.
2. GitHub 기존 파일을 새 파일로 전체 덮어씁니다.
3. Netlify에서 Deploy project without cache를 실행합니다.
4. 배포가 끝나면 Ctrl + Shift + R을 누릅니다.
5. PRO 7.7 SIMPLE 표시를 확인합니다.
