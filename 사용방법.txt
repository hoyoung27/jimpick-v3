짐픽 PRO 5.2 - 업체별 추가요금 직접 만들기

이번 핵심 작업
- 업체가 원하는 추가요금 항목 직접 생성
- 항목명과 금액 자유 입력
- 계단 작업, 장거리 운반, 주말, 야간, 주차 어려움 등 자유 설정
- 업체별 Supabase 서버 저장
- 견적마다 필요한 추가요금만 체크
- 선택한 추가요금 자동 합산
- 최종 견적은 대표가 직접 확정
- 견적서 공유 내용에 선택 추가요금 표시

사용 순서
1. 로그인 후 업체설정 클릭
2. '업체만의 추가요금 항목'에서 항목명과 금액 입력
3. 저장
4. 견적 결과에서 필요한 항목만 체크
5. 할인·추가조정 후 최종금액 적용

예시
- 계단 작업 100,000원
- 장거리 운반 50,000원
- 주말 작업 100,000원
- 야간 작업 150,000원

반드시 할 일
Supabase SQL Editor에서 supabase/setup.sql을 다시 실행하세요.

3. Commit changes
