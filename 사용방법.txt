짐픽 PRO 5.3 - 처음 요금 간편설정

이번 핵심 작업
- 신규 가입 업체에게 처음설정 화면 자동 표시
- 1단계: 업체명·대표자명·전화번호
- 2단계: 1톤·5톤 차량 기본요금
- 3단계: 사용할 추가요금 선택
- 큰 글씨와 큰 버튼
- 입력값 업체별 서버 저장
- 나중에 업체설정에서 수정 가능
- 설정을 건너뛰고 나중에 입력 가능

사용 순서
1. 로그인
2. 처음설정 화면에서 업체정보 입력
3. 차량요금 입력
4. 사용할 추가요금만 체크
5. 설정 완료
6. 바로 AI 견적 시작

기존 사용자는 상단 '처음설정' 버튼으로 다시 열 수 있습니다.

업로드
1. 압축 풀기
2. GitHub에 전체 파일과 netlify, supabase 폴더 업로드
3. Commit changes
4. Netlify에서 Deploy project without cache
5. Ctrl + Shift + R

정상 표시
build JIMPICK-PRO-5.3
