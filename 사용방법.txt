짐픽 PRO 6.2 BETA - 결과요약·출시판정

이번 핵심 작업
- 저장된 베타테스트 결과 목록
- 평균 완료율 계산
- 단계별 정상률 표시
- 문제와 불편사항 모아보기
- 테스트 수 부족 경고
- 출시 가능 / 보완 후 출시 / 출시 보류 자동 판정
- 베타테스트 결과 CSV 내려받기
- 기존 AI 견적·계약·문자·출시점검 기능 유지

사용 순서
1. 현장테스트를 여러 번 진행
2. 상단 '베타결과' 클릭
3. 단계별 정상률 확인
4. 기록된 문제점 확인
5. 출시 판정 확인
6. 필요하면 CSV 내려받기

출시 권장 기준
- 실제 현장 테스트 3회 이상
- 평균 완료율 95% 이상
- 각 주요 단계 정상률 90% 이상
- 기록된 중대한 문제 없음

업로드
1. 압축 풀기
2. GitHub에 전체 파일과 netlify, supabase 폴더 업로드
3. Commit changes
4. Netlify에서 Deploy project without cache
5. Ctrl + Shift + R
