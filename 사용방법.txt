짐픽 PRO 2.3 - 이사업체 전용 자동견적 엔진

이번 핵심 작업
- 품목별 예상 짐량 기준 적용
- 냉장고, 세탁기, 침대, 옷장, 소파 등 품목별 톤수 계산
- 무거운 품목 추가 반영
- 분해 필요 품목 추가 반영
- 특수 취급 품목 추가 반영
- 총 예상 짐량 자동계산
- 1톤 차량 자동계산
- 5톤 차량 자동계산
- 작업 인원 자동계산
- 예상 작업시간 자동계산
- 추천 견적금액 자동계산
- '이 자동견적을 적용하기' 버튼 추가

시험 순서
1. 로그인 없이 기능 시험하기
2. 고객정보 확인
3. 사진 촬영 또는 선택
4. AI 자동분석 시작
5. 품목확인 화면에서 자동견적 엔진 결과 확인
6. '이 자동견적을 적용하기' 클릭
7. 견적 결과와 문자 발송 확인

중요
현재 품목별 톤수와 금액은 초기 기준표입니다.
대표님의 실제 현장 경험을 반영해 품목별 기준을 계속 보정하면 더 정확해집니다.

업로드
1. 압축 풀기
2. GitHub에 전체 파일과 netlify, supabase 폴더 업로드
3. Commit changes
4. Netlify → Deploy project without cache
5. Ctrl + Shift + R

필수 환경변수
- SUPABASE_URL
- SUPABASE_ANON_KEY
- GEMINI_API_KEY

정상 표시
build JIMPICK-PRO-2.3
