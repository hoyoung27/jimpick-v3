짐픽 PRO 6.4 DEMO - 견적서 PDF·공유

추가 기능
- 고객용 견적서 미리보기
- 업체정보와 고객정보 표시
- 출발지·도착지 표시
- 차량·작업자·작업시간 표시
- 계산금액·할인·최종금액 표시
- 계약금·잔금 표시
- 추가요금 표시
- 고객서명 표시
- 인쇄 화면에서 PDF로 저장
- 휴대전화 공유 메뉴로 카카오톡 공유
- 공유 기능 미지원 시 견적내용 자동 복사

PDF 저장 방법
1. 자동견적 화면에서 'PDF로 저장' 클릭
2. 인쇄창이 열리면 프린터 선택에서 'PDF로 저장' 선택
3. 저장 버튼 클릭

업로드
1. 압축 해제
2. GitHub 전체 파일 덮어쓰기
3. Netlify Deploy project without cache
4. Ctrl + Shift + R
5. 왼쪽 위 PRO 6.4 DEMO 확인
