짐픽 PRO 5.4 - 견적내역·검색·재사용

이번 핵심 작업
- 저장한 견적 목록 보기
- 고객명 검색
- 전화번호 검색
- 출발지·도착지 주소 검색
- 이사 날짜 검색
- 이전 견적 다시 열기
- 같은 고객정보로 새 견적 시작
- 견적 삭제
- 업체별 자기 견적만 표시

사용 순서
1. 로그인 후 상단 '견적내역' 클릭
2. 고객명·전화번호·주소 또는 날짜로 검색
3. '견적 열기'로 이전 결과 확인
4. '재견적 시작'으로 고객정보를 불러와 새 견적 작성
5. 필요 없는 견적은 삭제

반드시 할 일
Supabase SQL Editor에서 supabase/setup.sql을 다시 실행하세요.

업로드
1. 압축 풀기
2. GitHub에 전체 파일과 netlify, supabase 폴더 업로드
3. Commit changes
4. Netlify에서 Deploy project without cache
5. Ctrl + Shift + R
