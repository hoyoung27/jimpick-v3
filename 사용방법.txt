짐픽 PRO 6.3.1 DEMO - 체험 버튼 긴급수정

수정 내용
- '로그인 없이 견적 화면 보기' 버튼 무반응 문제 수정
- 기존 프로그램 초기화 오류와 상관없이 독립적으로 실행
- 클릭하면 5단계 견적확인 화면으로 바로 이동
- 예시 고객, 차량, 인원, 시간, 금액 자동 표시

적용 순서
1. 압축을 풉니다.
2. GitHub에 전체 파일을 덮어씁니다.
3. Netlify에서 Deploy project without cache를 실행합니다.
4. 사이트에서 Ctrl + Shift + R을 누릅니다.
5. 왼쪽 위에 PRO 6.3.1 DEMO가 표시되는지 확인합니다.
6. 초록색 '로그인 없이 견적 화면 보기'를 누릅니다.
