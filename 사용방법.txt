짐픽 PRO 2.2 - 사진 AI 자동견적 현장시험

시험 순서
1. 로그인 화면에서 '로그인 없이 기능 시험하기'
2. 고객정보 확인 후 '다음: 사진 자동견적'
3. 사진 AI 화면에서 '지금 점검'
4. 로그인 서버와 사진 AI 서버가 정상인지 확인
5. 사진 촬영 또는 여러 장 선택
6. AI 자동분석 시작
7. 인식 품목, 1톤, 5톤, 인원, 시간, 금액 확인
8. '자동견적 결과 확인' 클릭
9. 견적 저장 또는 문자 발송 시험

이번 핵심 작업
- 사진 분석 결과표 추가
- 인식 품목 수 표시
- 1톤·5톤 차량 자동표시
- 인원·시간·금액 자동표시
- AI 실패 원인을 한글로 안내
- 견적 결과 화면으로 바로 이동

업로드
1. 압축 풀기
2. GitHub에 전체 파일과 netlify, supabase 폴더 업로드
3. Commit changes
4. Netlify → Deploy project without cache
5. Ctrl + Shift + R

필수 환경변수
- SUPABASE_URL
- SUPABASE_ANON_KEY
- GEMINI_API_KEY

정상 표시
build JIMPICK-PRO-2.2
