짐픽 PRO 6.9 DEMO - 현장안전·작업사진·수익분석 안정판

추가 기능
- 파손 위험·중량물 자동 분석
- 안전/주의/위험 표시
- 작업 전 체크리스트
- 작업 전·작업 후 사진 저장
- 대표 전용 메모
- 작업자 인건비·차량비·유류비·식대·통행료·기타비용 입력
- 총매출·총비용·예상 순이익·순이익률 계산
- 짐픽 현장 점수 표시
- 예상 작업시간 추천은 포함하지 않음

안정성 확인
- 추가 JavaScript 문법 검사 완료
- 전체 인라인 JavaScript 문법 검사 완료
- Netlify 함수 문법 검사 완료
- 중복 화면 ID 검사 완료
- 기존 기능과 독립적으로 작동하도록 구성

업로드
1. 압축을 풉니다.
2. GitHub 기존 파일을 새 파일로 전체 덮어씁니다.
3. Netlify에서 Deploy project without cache를 실행합니다.
4. 사이트에서 Ctrl + Shift + R을 누릅니다.
5. PRO 6.9 DEMO 표시를 확인합니다.
